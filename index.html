Conhecendo o GITHUB

Realizando COMMITS

  • Utilizar nomes de Commits objetivos e curtos até 50 caracteres
  • Os commits são "cópias" de nosso projeto naquele momento
  • O campo descrição irá ajudar futuramente a entender o motivo daquele commit, utilize sempre que achar necessário
  • Iniciar com letra maiúscula
  • Não terminar com ponto final
  • Colocar em forma imperativa. Ex: Como se estivesse dando uma ordem. Remover o banner rotativo
  • Cuidado com o idioma, utilizar pt-br ou en-us

Dicas de boas práticas para nomes de commits

  • Adicionar: Funcionalidade zoom da galeria de imagens adicionada
  • Alterar: Funcionalidade zoom da galeria de imagens alterada
